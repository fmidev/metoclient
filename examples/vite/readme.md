# MetOClient with Vite

This example demonstrates how the [`metoclient` package](https://www.npmjs.com/package/@fmidev/metoclient) can be used with Vite.

Install the project dependencies.

    cd examples/vite
    npm install

Start the development server with hot module replacement.

    npm run dev

Build for production.

    npm run build

Preview the production build locally.

    npm run preview
